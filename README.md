# 🎨 Мульти-Пульти (Multi-Pulti)

Простая, яркая и увлекательная веб-студия для создания покадровой анимации и рисунков прямо в браузере. Идеально подходит для детей, начинающих художников и всех, кто хочет быстро создать забавный мультик!

## ✨ Особенности

* 🖌️ **Инструменты для рисования:** Кисть, заливка и ластик с удобной настройкой размера.
* 🎞️ **Покадровая анимация:** Создавайте мультики кадр за кадром. Удобная временная шкала (таймлайн), добавление, копирование и удаление кадров.
* ⏩ **Настройка воспроизведения:** Просмотр анимации в реальном времени с возможностью изменения скорости (от 1 до 24 FPS).
* 🎨 **Продвинутая палитра:** Выбор любого цвета с помощью удобного ползунка, а также автоматическое сохранение недавних и любимых цветов.
* ⭐ **Стикеры:** Огромный набор эмодзи-стикеров, которые можно добавлять на холст, перемещать и менять их размер.
* 🦋 **Умные режимы:** 
  * **Симметрия:** Рисуйте красивые симметричные узоры (бабочки, снежинки).
  * **Помощник:** Помогает рисовать ровные линии.
* 🖼️ **Фоны и шаблоны:** Загружайте свои картинки или используйте готовые шаблоны для раскрашивания.
* 💾 **Автосохранение:** Случайно закрыли вкладку? Не беда! Все ваши кадры, история действий и любимые цвета автоматически сохраняются в браузере.
* 📥 **Экспорт:** Сохраняйте свои шедевры как статичные картинки (PNG) или делитесь полноценными анимированными мультиками (GIF)!

## 🚀 Технологии

Проект построен с использованием современных веб-технологий:
* **React 18** — UI библиотека
* **TypeScript** — строгая типизация
* **Vite** — быстрый сборщик
* **Tailwind CSS** — стилизация интерфейса
* **HTML5 Canvas** — движок для рисования и рендеринга
* **gif.js** — генерация GIF-анимаций на стороне клиента

## 🛠️ Установка и запуск локально

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/Moii-gh/Multi-Pulti.git
   cd multi-pulti
   ```

2. Установите зависимости:
   ```bash
   npm install
   ```

3. Запустите сервер для разработки:
   ```bash
   npm run dev
   ```

4. Откройте браузер и перейдите по адресу `http://localhost:3000` (или другому порту, указанному в консоли).

## 🎮 Как пользоваться

1. **Рисование:** Выберите инструмент (кисть, заливка) на левой панели, выберите цвет и начните рисовать на белом холсте.
2. **Анимация:** Нажмите кнопку `+` на нижней панели, чтобы добавить новый кадр. Рисунок с предыдущего кадра останется полупрозрачным (эффект кальки), чтобы вам было удобнее рисовать продолжение движения.
3. **Воспроизведение:** Нажмите зеленую кнопку "Играть", чтобы увидеть свой мультик.
4. **Сохранение:** Нажмите на иконку дискеты (GIF), чтобы скачать готовую анимацию на свой компьютер.
